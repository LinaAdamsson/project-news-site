CLOUD CROWD

Your guide to cloudspotting. Clouds are grouped into 10 different genera, organized based on shape and the altitude where they’re found. The genera names are all formed from the same five latin terms — cirro, cumulo, strato, nimbo, alto — which are mixed and matched to create names like cumulonimbus, cirrostratus.
